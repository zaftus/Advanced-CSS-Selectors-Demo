# Advanced CSS Selectors Demo

## Overview
This project demonstrates the use of advanced CSS selectors to target specific elements on a webpage. It covers pseudo-classes, pseudo-elements, combinators, attribute selectors, and more. The project is designed as a learning resource for developers aiming to improve their CSS skills.

## Features
- Multiple HTML pages with consistent layout
- Modular CSS files targeting specific elements
- Advanced selectors including:
  - Pseudo-classes (`:hover`, `:first-child`, `:nth-child`, etc.)
  - Pseudo-elements (`::before`, `::after`)
  - Combinators (`>`, `+`, `~`, ` `)
  - Attribute selectors (`[type="text"]`, `[href^="https"]`, `[href$=".html"]`)
- Responsive layout using Flexbox and Grid
- Buttons, cards, tables, modals, tooltips, and animations

## File Structure
- `index.html`, `about.html`, `contact.html`, `services.html` — main pages
- `css/` — modular CSS files for layout, typography, buttons, tables, forms, animations, etc.
- `js/` — JavaScript files for basic interactions and animations
